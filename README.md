## 🛡️ Security Criteria Breakdown

<div align="center">

| Criteria | Requirement | Why It Matters |
|:---:|:---:|:---|
| 📏 **Length** | ≥ 8 characters | Longer = exponentially harder to brute-force |
| 🔠 **Uppercase** | At least one `A-Z` | Adds unpredictability to the character set |
| 🔡 **Lowercase** | At least one `a-z` | Baseline for structural complexity |
| 🔢 **Numbers** | At least one `0-9` | Defeats simple dictionary attacks |
| ✨ **Special Chars** | At least one `!@#$%^&*` | Maximum entropy against all attack vectors |

</div>
